## About The Project
The project addresses simultaneous Localisation and Mapping (SLAM) of a [e-puck robot](http://www.e-puck.org/) by using the robot's rotary encoders of the wheel, the robotâ€™s IMU and multiple Infrared sensors around the robot. This information is then sent to a PC to visualise it.
